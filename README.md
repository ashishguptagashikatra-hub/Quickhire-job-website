QuickHire is a responsive Job Offering Landing Page built using **HTML, CSS, and JavaScript**.

## Features
- Responsive navbar with mobile menu
- Job listing cards
- Benefits section
- Job application form
- Consent popup for data usage permission
- Clean UI and mobile-friendly design

## Files Included
- `index.html` (Main website page)
- `style.css` (Website styling)
- `script.js` (Popup + mobile menu logic)
- `README.md` (Project description)

## How to Run Locally
1. Download this repository or clone it
2. Open `index.html` in any browser

## Host on GitHub Pages
1. Upload all files to a GitHub repository
2. Go to **Settings**
3. Go to **Pages**
4. Under Source select:
   - Branch: `main`
   - Folder: `/root`
5. Save
6. Your site will be live in a minute

## Note
This project does not store user data in any database.
It only displays the form details in an alert after user consent.
